<h1>Notification Service</h1>
Projeto realizado no evento Ignite Lab da RocketSeat onde realizamos um Back-end de um Microsserviço Escalável de Notificações.

<br>
<h2>Tecnologias Usadas</h1>
<li>Typescript</li>
<li>Nestjs</li>
<li>Nodejs</li>
<li>Prisma</li>
<li>Jest</li>
<br>
<li>Neste projeto foram aplicados conceitos de: Inversão e Injeção de Dependência, Escalabilidade, Domain Driven Design, Testes Unitários, Desacoplamento de Camadas e Arquitetura Limpa</li>
